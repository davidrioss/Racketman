# How to install xmake
### Windows
Invoke-Expression (Invoke-Webrequest 'https://xmake.io/psget.text' -UseBasicParsing).Content
xmake f -p mingw --mingw="C:\MinGW"
### Linux
curl -fsSL https://xmake.io/shget.text | bash
